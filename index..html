Fixed Container

This content is inside a fixed width container. It stays centered and is best for readable layouts like blogs or portfolios.

Fluid Container

This section spans the full width of the screen. Perfect for banners, headers, or sections that need maximum space.

Container with Padding

Container with Border and Color

.container-sm → Full width until small screens, then fixed. Good for mobile layouts.

.container-md → Full width until medium screens, then fixed. Useful for tablets.

.container-lg → Full width until large screens, then fixed. Best for desktops.

.container-xl → Full width until extra-large screens, then fixed. Great for wide monitors.

.container-xxl → Full width until extra-extra-large screens, then fixed. Perfect for ultra-wide displays.
